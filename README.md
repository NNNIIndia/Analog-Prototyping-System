# DEPRECATED
## Please See (and use): [Mark's Manhattan Toolkit](https://github.com/macaba/TheManhattanToolkit)
After getting a lot of helpful feedback, it was determined that the SMD adapters had a tendency to short out between adjacent pads easily, require excessive solder and accumulate flux and flux residue. The copper wire connection method was neither straightforward nor particularly EMI immune, and was theorized to worsen parasitic effects.
